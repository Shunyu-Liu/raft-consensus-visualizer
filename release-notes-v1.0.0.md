# Raft Explorer v1.0.0

Raft Explorer is an interactive educational simulator for understanding the Raft consensus algorithm.

## Highlights

- Deterministic five-node Raft simulation
- Leader election
- RequestVote
- Heartbeats as empty AppendEntries
- Log replication
- Majority commit
- Crash and restart
- Leader re-election
- Split vote
- Network partition
- Minority cannot commit
- Conflicting logs
- nextIndex backtracking
- Log reconciliation
- Learn page
- Glossary
- Raft safety explanations
- Responsive and accessible UI
- GitHub Pages deployment

## Interactive Scenarios

- Basic Leader Election and Log Replication
- Leader Failure and Re-election
- Split Vote
- Network Partition - Minority Leader and Majority Progress
- Conflicting Logs and Log Reconciliation

## Documentation

- Architecture
- Raft Model
- Scenario Guide
- Roadmap
- Contributing Guide

## Limitations

- No Snapshot
- No Log Compaction
- No Membership Change
- No Joint Consensus
- No PreVote
- No CheckQuorum
- No real network transport
- No durable Raft storage
- Not a production Raft implementation
