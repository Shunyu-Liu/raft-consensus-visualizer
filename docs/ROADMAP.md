# Roadmap

Raft Explorer currently focuses on the core teaching path for leader election, log replication, failures, partitions, and conflicting log repair.

## Out of Scope for 1.0.0

- Snapshot
- InstallSnapshot
- Log compaction
- Membership changes
- Joint consensus
- Leader transfer
- PreVote
- CheckQuorum
- ReadIndex
- Linearizable reads
- Durable disk
- Real network transport
- Production Raft

## Possible Future Work

- Optional snapshot and log compaction teaching scenario
- Membership change and joint consensus guide
- More visual regression coverage
- Real screenshot gallery
- Additional accessibility audits with external tooling
