# Security Policy

Raft Explorer is a static educational web application. It does not include a backend, database, authentication system, telemetry, or secret-bearing API integration.

## Supported Versions

The current maintained version is `1.0.0`.

## Reporting a Vulnerability

Please open a private report through GitHub Security Advisories if the repository is hosted on GitHub. If advisories are not enabled, open an issue without including exploit details or secrets.

Include:

- Affected file or workflow
- Reproduction steps
- Expected impact
- Suggested fix, if known

Do not include real credentials, tokens, or private keys in reports.
